Présentation

TransportCollectifs est une application mobile de covoiturage qui connecte les conducteurs et les voyageurs se rendant vers la même destination.
L’objectif : réduire les coûts, faciliter le transport entre villes et aider ceux qui n’ont pas de voiture en leur permettant de trouver un conducteur disponible pour leur trajet.

Le projet offre une solution moderne, intuitive et accessible pour encourager le partage de trajets et optimiser les déplacements quotidiens.

 Objectifs du projet

Mettre en relation conducteurs et voyageurs.

Réduire les coûts de déplacement.

Faciliter le transport dans les zones où il y a peu de services publics.

Offrir un système simple, sécurisé et basé sur la confiance.

Créer une alternative locale aux services comme Amigo ou BlaBlaCar.

 Rôles des utilisateurs
 Passager

Rechercher un trajet selon la destination, date et heure.

Réserver une place auprès d’un conducteur.

Voir les infos du conducteur (nom, véhicule, prix).

Recevoir une confirmation du trajet.

 Conducteur

Publier un trajet : point de départ, destination, horaire.

Indiquer le nombre de places disponibles.

Gérer les réservations des passagers.

Confirmer ou refuser les demandes.

 Administrateur 

Superviser les utilisateurs

Vérifier les annonces

Gérer les litiges ou signalements

 Fonctionnalités principales

Recherche de trajets en temps réel

Système de réservation simple

Profil conducteur + infos du véhicule

Cartes ou liste des trajets disponibles

Interface moderne et adaptée mobile

Notification ou message de confirmation
